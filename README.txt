長照金額計算程式
================

【功能說明】
讀取長照服務原始檔（Excel），依據 B 碼服務紀錄，
自動計算每位居服員的 A 碼補助款，並輸出完成版 Excel。

計算規則：
  補助款 = 服務項目給付價格 × 服務天數 ÷ 當日服務人數 ÷ 2
  （÷2 為公司抽成）

輸出結果放在 output\ 資料夾，檔名含日期時間，不會覆蓋舊檔。


【第一次使用：安裝環境】

1. 確認電腦已安裝 Python 3.8 以上
   （下載：https://www.python.org/downloads/
    安裝時請勾選 "Add Python to PATH"）

2. 雙擊執行 install.bat
   等待畫面顯示「安裝完成」後關閉視窗
   （之後不需要再執行此步驟）


【每次使用】

1. 將原始檔（.xlsx）放入本資料夾
2. 雙擊執行 run.bat
3. 若資料夾內只有一個 xlsx，程式會直接處理
   若有多個 xlsx，程式會列出選單，輸入編號後按 Enter
4. 處理完成後，至 output\ 資料夾取得結果檔案


【注意事項】

- 原始檔需包含以下兩個工作表：
    「中央服務紀錄(B碼)+姓名」
    「A碼項目清冊」

- 若出現 [!][待確認] 的警告，表示該筆資料在 B 碼中
  找不到對應的居服員紀錄，輸出檔中已預留空白位置，
  請人工開啟 output\ 內的結果檔，填入正確數量與金額。


【資料夾結構】

  程式資料夾\
  ├── process.py       程式主體
  ├── run.bat          執行程式（雙擊此檔）
  ├── install.bat      第一次安裝環境用
  ├── README.txt       本說明文件
  ├── 原始檔.xlsx      ← 將要處理的 Excel 放在這裡
  └── output\          ← 輸出結果會自動放在這裡
        └── 原始檔_完成_20260306_1430.xlsx
